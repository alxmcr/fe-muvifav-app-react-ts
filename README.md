
# Muvifav

An application to display about movies information.


## Run Locally

Clone the project

```bash
  git clone https://github.com/alxmcr/fe-muvifav-app-react-ts
```

Go to the project directory

```bash
  cd fe-muvifav-app-react-ts
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
